Problema:Leia N números inteiros do usuário, depois da leitura, imprima todos os números na ordem inversa de leitura

  • Solução Usando Listas
    • Leia o valor de N do usuário
    • Declare uma lista L com N posições
    • Leia cada número e armazene em L
    • Imprima os valores de L começando do último até o primeiro

Código da Solução com Listas


                L = []
                for i in range(N):
                  var = int(input("valor?"))
                  L.insert(i,var)
                for i in range(N):
                  print(L[N-1-i])

  • Listas genéricas ex: ["banana", 2, ["outra lista", 2, 3.0], 5]

Problema: Leia do usuário um número N e gere aleatóriamente um conjunto com N numeros entre 0 e 20, depois imprima os números do conjunto na ordem crescente


                  import random

                  #troca os elementos: vetor[pos1] = vetor[pos2] e vetor[pos2]=vetor[pos1]
                  def trocaElementos(pos1, pos2, vetor):
                    aux = vetor[pos1]
                    vetor[pos1] = vetor[pos2]
                    vetor[pos2] = aux

                  N = int(input("Qual o Tamanho?"))

                  vetor = []

                  for i in range(N):
                    vetor.insert(i, random.randint(0,20))

                  print("Vetor desordenado")
                  print(vetor)

                  for i in range(N):
                    elemento = vetor[i]
                    for j in range(i,N): #ordena a posição atual do vetor
                      if elemento > vetor[j]: #testa se o elemento atual é menor do que os outros do vetor
                        elemento = vetor[j] #quarda o menor elemento como sendo o elemento atualmente testado
                        trocaElementos(i,j,vetor)

                  print("Vetor ordenado")
                  print(vetor)
