Linguagem Python

Descrição básica
  • Criada em 1991 por Guido Van Rossum
  • Linguagem interpretada ou compilada
  • Suporta múltiplos paradimgas de programação como o Orientado à Objetos e Estruturado
  • Também conhecida como um tipo de linguagem de "script", pois foi concebida para substituir a linguagem Shell dos sistemas linux
Características Básicas
  • Fortemente orientada à "espaços em branco/tabs"
  • Dinamicamente tipada
  • Usada em vários contextos como: Inteligencia Artificial, Ciencia de Dados e Sistemas Web

Comandos Básicos Python

Elementos básicos da sintaxe até agora

  • Linhas que começam com "#"(comentários) são ignoradas pelo computador
  • Expressões podem ou não termiar em ";"
  • Valores textuas sempre são escritos entre aspas simples ou duplas
  • Para chamar funções sempre usamos "(" e ")" com os parâmetros separados por virgula
  • Poemos usar "(" e ")" para controlar a ordem que as expressões são executadas
